# EMPLOYEE ATTRITION PREDICTION
# OBJECTIVE
To analyze employee data and build a classification model to predict whether an employee will leave the organization.
# TOOLS USED
Python → main programming language
Jupyter Notebook → for coding and analysis
Libraries 
Pandas → data cleaning & preprocessing
NumPy → numerical operations
Matplotlib → basic data visualization
Seaborn → advanced visualization
Scikit-learn → machine learning models
# KEY INSIGHTS
- High attrition in employees with low job satisfaction
